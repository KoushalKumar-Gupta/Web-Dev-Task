## 🚀 Repository Structure

### 📁 [Task-01-Basics](./Task-01-Basics)
* **Focus:** Core building blocks of modern web pages and introducing logical scripting.
* **Tech Stack:** HTML5, CSS3, JavaScript (Basics)
* **Key Concepts:**
  * Semantic HTML elements and clean layout structuring.
  * Basic styling, padding, margins, and colors.
  * Basic JS: Variables, simple functions, console logging, or basic alerts.

### 📁 [Task-02-Intermediate](./Task-02-Intermediate)
* **Focus:** Dynamic behavior, responsive design, and deep dive into interactivity.
* **Tech Stack:** HTML5, CSS3 (Advanced), JavaScript (DOM)
* **Key Concepts:**
  * **Advanced CSS:** Creating flexible layouts using Flexbox and CSS Grid.
  * **Responsiveness:** Using Media Queries to make it mobile-friendly.
  * **JavaScript:** Event listeners, dynamically updating the UI, and handling user input.
